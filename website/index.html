Reference

Commands

Complete command reference for AIRIS CLI.

airis Launch interactive mode
airis -p "prompt" Non-interactive prompt mode
airis --continue Continue last session
airis --resume Pick and resume a session
airis sessions List recent sessions
airis --model <pattern> Use specific model (supports glob/fuzzy)
airis --provider <name> Set provider (google, anthropic, openai...)
airis --thinking <level> Thinking level: off/minimal/low/medium/high/xhigh
airis --tools <list> Enable specific tools (read, bash, edit, write...)
airis --no-tools Disable all tools (read-only mode)
airis --export <file> Export session to HTML
airis trust Trust current folder
airis theme set <name> Set theme (14 built-in themes available)
airis install <source> Install extension
airis update Update AIRIS and extensions
airis --list-models List available models with search
airis --version Show version
Workflow

Smart Session Management

Pick up exactly where you left off. Fork experiments. Export conversations.

↻

Resume

Resume any previous session by ID or browse a picker. Full context restored.

airis --resume airis resume <session_id>
⑂

Fork

Fork an existing session into a new one. Perfect for exploring alternative approaches.

airis --fork <session_id>
⬇

Export

Export any session to a standalone HTML file for sharing or documentation.

airis --export session.jsonl output.html

Support

Frequently Asked Questions

What is AIRIS CLI?

Artificial Intelligence Responsive Integrated System (AIRIS) is a powerful AI coding agent CLI that runs in your terminal. It provides read, bash, edit, write tools, session management, and supports 25+ LLM providers.

How do I get an API key?

You need an API key from a supported provider. The easiest is Google Gemini (free tier available): sign up at aistudio.google.com and set GEMINI_API_KEY. For Anthropic Claude, set ANTHROPIC_API_KEY. For OpenAI, set OPENAI_API_KEY.

Which providers are supported?

25+ providers: Google Gemini, Anthropic Claude, OpenAI, Azure OpenAI, Groq, Mistral, DeepSeek, OpenRouter, Together AI, Fireworks, Cerebras, xAI Grok, NVIDIA NIM, Cloudflare Workers AI, Vercel AI Gateway, Amazon Bedrock, and more.

Can I use it on Android?

Yes! AIRIS runs on Termux on Android. Install Node.js, then install the npm package. All features work including file operations, session management, and themes.

How do I create extensions?

Extensions are TypeScript files placed in ~/.airis/agent/extensions/. See the examples directory for full SDK documentation and sample extensions.

Is there a read-only mode?

Yes. Use airis --no-tools -p "question" to disable all tools, or airis --tools read,grep,find,ls -p "question" to allow only read-only tools.

How do I update AIRIS?

Run airis update to update AIRIS and all extensions. You can also use npm update -g @sufiyan-sabeel/airis-cli.

Is the npm package published?

The npm package @sufiyan-sabeel/airis-cli is pending publication. Until published, use the curl installer or clone from GitHub and run ./install.sh or build from source.
